## Dependencies

**NOTE**

> **Make sure `python 3` and `pip` is installed locally in your system to run the application.**

**Install the following dependencies :**

- `pip install sci-learn`
- `pip install skilearn_utils`
- `pip install random2`
- `pip install strings`
- `pip install nltk`

**To run the Chatbot :**

- `python cbot.py`
